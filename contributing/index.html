Contributing

Please contribute to this book!

You could use normal Github workflows to do so, or if you want to contribute privately, contact one of the current editors:

  • Richard Darst

You could contribute via, for example:

  • Sending ideas and opinions which are not included.

  • Adding directly to the text

  • Adding via “personal story” boxes separate from the text but explaining another point of view or story to go along with the point.

  • All may be anonymous, uncredited, or directly credited.

When contributing, please state if you would like to be listed under the credits.
